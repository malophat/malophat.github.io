const app = express(); 用于创建一个 Express 应用程序实例。

app.use() 和 app.set() 是用于配置 Express 应用程序的方法。

app.use() 是用来注册中间件函数的，它将中间件函数附加到应用程序对象的请求处理管道中，用于处理请求。

app.set() 是用于设置应用程序变量的，它可以设置应用程序的配置选项，例如视图引擎（可以res.render(“.pug”)）、视图路径、端口号等。

具体来说，app.use() 用于引入中间件，例如解析请求体、处理 cookie、设置静态文件目录等。app.set() 用于设置应用程序的全局变量，例如视图引擎（视图抽象层）、应用程序端口、请求超时时间等。

需要注意的是，app.set() 和 app.use() 方法都是应用程序级别的方法，它们设置的参数或中间件将应用于整个 Express 应用程序，而不是特定的路由或控制器。
